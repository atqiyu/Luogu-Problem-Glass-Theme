# Luogu-Problem-Glass-Theme

这是一个名为 **“Luogu Problem Glass Theme” (洛谷题目页毛玻璃动漫风美化)** 的 **Tampermonkey (油猴) 用户脚本项目**。

### 1. 项目核心目标
该项目旨在彻底重构洛谷（Luogu）题目详情页的视觉体验，将原本朴素、传统的界面改造为具有**现代感、通透感和二次元动漫风格**的“毛玻璃（Glassmorphism）”界面。它特别针对侧边栏信息布局进行了极致优化，以满足用户对紧凑排版和精准色彩控制的需求。

### 2. 主要功能特性

#### 🎨 视觉美化 (Visual Aesthetics)
*   **毛玻璃特效**：对所有主要卡片（头部横幅、侧边信息卡、题目描述区）应用 `backdrop-filter: blur()` 和高透明度背景，营造轻盈、通透的层次感。
*   **动漫风配色**：使用柔和的粉紫、淡蓝、薄荷绿等渐变色作为卡片背景，配合径向光晕装饰，符合动漫审美。
*   **渐变标题**：题目主标题采用线性渐变字体，增强视觉冲击力。
*   **自定义壁纸支持**：内置设置面板，允许用户上传本地图片作为页面背景，并自动保存至浏览器本地存储。

#### 📐 极致紧凑布局 (Extreme Compact Layout)
*   **2x2 网格系统**：将侧边栏前四个信息项（题目编号、提供者、难度、历史分数）强制重组为 **CSS Grid 2x2 矩阵**。
*   **零间距优化**：
    *   消除标签（Label）与数值（Value）之间的垂直空隙，行高压缩至 `1.0`。
    *   消除卡片内部的多余 Padding，使内容极度紧凑。
    *   底部三个功能按钮（提交记录、查看题解、题目反馈）实现**无间隙并排**，视觉上形成一个整体条块。

#### 🛠️ 功能修复与增强 (Fixes & Enhancements)
*   **精准难度颜色映射**：摒弃了不稳定的样式继承，采用**文本匹配硬编码**方式。根据难度文字（如“普及+/提高−”）直接注入用户指定的 RGB 颜色值，确保颜色 100% 准确且不受网站更新影响。
*   **条纹彻底清除**：针对顶部导航栏（题目描述/提交答案按钮），强制移除所有原生背景图、边框和阴影，解决“去不掉的条纹”问题。
*   **无关内容隐藏**：自动隐藏“相关讨论”、“推荐题目”及广告模块，保持页面整洁。

### 3. 技术实现亮点

*   **DOM 劫持与重绘**：通过 `MutationObserver` 监听页面变化，确保在洛谷这种 SPA（单页应用）架构下，路由跳转或动态加载后样式依然生效。
*   **CSS 优先级战争**：大量使用 `!important` 和更具体的选择器路径，以覆盖洛谷原生复杂的 Vue 组件样式。
*   **本地化存储**：利用 `localStorage` 保存用户上传的 Base64 壁纸数据，实现持久化个性化配置。
*   **非侵入式注入**：脚本通过动态创建 `<style>` 标签注入 CSS，不修改原始 HTML 结构，仅通过添加类名（如 `.lg-decorated`）进行样式挂钩。

### 4. 适用人群
*   喜欢**二次元/动漫风格**界面的洛谷用户。
*   对页面**信息密度**有极高要求，希望一眼看清关键数据（难度、编号等）的用户。
*   对原生界面细节（如条纹背景、颜色偏差）感到不满的强迫症用户。

### 5. 当前版本状态 (v0.9)

这个项目本质上是一个**前端 UI 覆盖层（UI Overlay）**，它在不改变网站功能逻辑的前提下，通过 CSS 和少量 JS DOM 操作，为用户提供了一个高度定制化的视觉皮肤。
